ParseDecimal32 PROC

   Converts an unsigned decimal integer string into 32-bit binary..

Call args:  EDX = offset of decimal string
            ECX = length of decimal string

Return arg: EAX = binary value
                 (See the ReadDec procedure for information about how
                  the Carry flag is affected.)

Example:

.data
buffer BYTE "8193"
bufsize = ($ - buffer)
decval DWORD ?

.code
    mov   edx,OFFSET buffer
    mov   ecx,bufsize
    call  ParseDecimal32
    mov   decval,EAX
